Step 4: Manual Extension Check

Go to chrome://extensions/ and check:

  • Is "Developer mode" enabled?
  • Is "GhostScan Privacy Tool" listed?
  • Is it enabled (toggle is ON)?
  • What is the extension ID shown?
  • Are there any error messages?

Step 5: Load Extension (if not loaded)

If the extension is not listed:

  1. Go to chrome://extensions/
  2. Enable "Developer mode"
  3. Click "Load unpacked"
  4. Navigate to: GhostScan-Commercial/apps/extension/dist
  5. Select the dist folder
